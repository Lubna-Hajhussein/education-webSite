Meta && Special-Characters Advanced
\dit matches any digit character ( same as [0-9] )
\wit matches any word character ( a-z , A-Z , 0-9 And _ )
\sit matches a whitespace character ( spaces, tabs )
\tit matches a tab character only
?The zero or one quantifier (makes a preceding char optional)
.Any character whatsoever (except the newline character)
*The 0 or more quantifier (a bit like +)
+The one or more quantifier
\Escapes the normal behavior of some characters and turn them into a meta characters with special abilities
[ ]The character set
[ ^ ]The negate symobol in a character set
